# 実行計画

## フェーズ構成

| フェーズ | 内容 | 実行 |
|---------|------|------|
| Phase 1: タスク取得 | MCP API で Todoist プロジェクトのタスクを取得し、見積時間をパース | ✅ |
| Phase 2: スロット生成 | 作業可能時間帯（ランチ除外）を 15 分スロット列として生成 | ✅ |
| Phase 3: 最適化 | ortools CP-SAT でタスク→スロット割り当てを最適化 | ✅ |
| Phase 4: API 更新 | update-tasks で dueString を設定 | ✅ |
| Phase 5: レポート出力 | 結果と警告を標準出力/stderr に表示 | ✅ |

## 成果物

- `scheduler.py` — メインスクリプト（`main.py` とは独立したファイル）
- 既存 `main.py` は変更しない

## スキップするフェーズ
