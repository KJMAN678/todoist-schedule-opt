# 要件定義: todoist-scheduler

## 機能要件（EARS形式）

### タスク取得

- **REQ-1**: `--project-id` で指定したプロジェクトID（デフォルト: `6gc6X66RQH7Qcmxh`）に属するタスクを Todoist MCP API の `find-tasks` ツールで取得しなければならない。
- **REQ-2**: 取得するタスク数の上限は 100 件とする。

### 見積時間パース

- **REQ-3**: 各タスクの `description` フィールドを float として解釈し、時間単位の見積作業時間とする。
- **REQ-4**: `description` が空文字、空白のみ、または数値として解釈できない場合、そのタスクの ID・タスク名・理由を stderr に出力し、スケジュール対象から除外して処理を継続しなければならない。

### スロット生成

- **REQ-5**: `--start`（HH:MM）〜 `--end`（HH:MM）の時間帯を 15 分単位のスロット列として生成しなければならない。
- **REQ-6**: `--lunch-start` と `--lunch-end` が両方指定された場合、その時間帯に該当するスロットを利用不可として除外しなければならない。
- **REQ-7**: `--lunch-start` と `--lunch-end` はどちらか片方のみの指定を許可せず、両方指定または両方省略とする。
- **REQ-8**: 各タスクの所要スロット数は `ceil(duration_hours × 60 / 15)` とする。

### 最適化

- **REQ-9**: ortools CP-SAT ソルバーを用いてタスクを利用可能スロットに割り当てる。
- **REQ-10**: 各タスクは連続した所要スロット数分のスロットを占有しなければならない。
- **REQ-11**: 複数タスクのスロット範囲が重複してはならない。
- **REQ-12**: タスクの割り当て順序は `find-tasks` の取得順に従わなければならない（タスク i の開始スロット < タスク i+1 の開始スロット）。
- **REQ-13**: 割り当て可能なタスク数を最大化することを目的とする。
- **REQ-14**: 利用可能スロットに収まらないタスクは未スケジュールとし、タスク名と理由を stderr に警告出力する。

### API 更新

- **REQ-15**: スケジュール済みタスクに対し、`update-tasks` ツールで `dueString` を `"YYYY-MM-DD HH:MM"` 形式（例: `"2026-05-09 09:00"`）で設定しなければならない。
- **REQ-16**: 未スケジュールタスクの `dueString` は変更してはならない。
- **REQ-17**: `--date` 引数が省略された場合、スクリプト実行日の日付を使用する。

### レポート出力

- **REQ-18**: スケジュール済みタスクの一覧（タスク名・開始時刻・見積時間）を標準出力に表示しなければならない。
- **REQ-19**: エラー・警告はすべて stderr に出力し、標準出力と分離しなければならない。

---

## 非機能要件

- **NFR-1**: Python 3.13、`uv run --env-file .env.local python scheduler.py` で実行可能であること。
- **NFR-2**: 認証は `TODOIST_API_TOKEN` 環境変数を使用する。
- **NFR-3**: MCP API エラーは例外を再送出してスタックトレースを表示する。
- **NFR-4**: CLI 引数不正時は argparse がエラー表示して exit(2) する。

---

## CLI 引数仕様

| 引数 | 必須 | デフォルト | 説明 |
|------|------|-----------|------|
| `--project-id` | 任意 | `6gc6X66RQH7Qcmxh` | 対象プロジェクト ID |
| `--start` | **必須** | — | 作業開始時刻（HH:MM） |
| `--end` | **必須** | — | 作業終了時刻（HH:MM） |
| `--lunch-start` | 任意 | — | ランチ開始時刻（HH:MM） |
| `--lunch-end` | 任意 | — | ランチ終了時刻（HH:MM） |
| `--date` | 任意 | 実行日 | スケジュール対象日（YYYY-MM-DD） |
